# 🎨 Aakarshan – Custom Canvas Art Store

**Aakarshan** is a full‑stack web application built to sell custom canvas art and paintings online.  
It combines a modern storefront for customers with a powerful admin panel for administrators, delivering a seamless experience end‑to‑end.

---

## 🌐 Live Demo
- **Frontend (Netlify)**: https://aakarshann.netlify.app/  
- **Backend (Render)**: https://aakarshan.onrender.com  

---

## 🔹 Key Features
- **Modern Storefront** → Browse and purchase custom canvas art and paintings with a clean, responsive UI.  
- **Admin Panel** → Manage inventory, add new artworks, update product details, and track orders.  
- **Secure Authentication** → User login and admin access with JWT authentication.  
- **Order Management** → Customers place orders; admins can view and update order statuses.  
- **Responsive Design** → Optimized for desktop and mobile with React + SCSS.  
- **Scalable Backend** → Node.js + Express.js APIs connected to MongoDB for robust data handling.  

---

## 🔹 Tech Stack
- **Frontend**: React, SCSS, Material UI  
- **Backend**: Node.js, Express.js  
- **Database**: MongoDB  
- **Other Tools**: JWT Authentication, Nodemon, SweetAlert
